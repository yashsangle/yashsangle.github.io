Project Overview

The University Office of Planning and Budgeting (UOPB) has implemented an automated solution to transform manual data processes into efficient, AI-powered workflows. Our system eliminates manual PDF extraction and web data retrieval through advanced automation, providing instant access to institutional data through natural language queries.

Explore Features

Key Features

Automated Data Extraction

Advanced PDF table extraction using Tabula and web data collection via Selenium eliminates manual data entry.

BigQuery Data Warehouse

All institutional data is stored in Google BigQuery for fast querying and analysis with enterprise-grade security.

AI-Powered Chatbot

Dialogflow CX chatbot provides natural language access to institutional data with contextual understanding.

Interactive Dashboards

Real-time visualization tools connected directly to live data sources for immediate insights.

Technology Stack

Python (Pandas, Tabula, Selenium) Google BigQuery Google Cloud Platform Dialogflow CX Google Cloud Armor OAuth 2.0

Security & Compliance

  • OAuth 2.0 authentication for all services
  • Data encryption at rest and in transit
  • Role-Based Access Control (RBAC)
  • Regular security audits and monitoring
